# 트리(tree) 알아보기

##### 트리의 특징
- 순환 구조(cycle)를 지니지 않고, 1개의 루트 노드가 존재합니다.
- 루트 노드를 제외한 노드는 단 1개의 부모 노드를 갖습니다.
- 트리의 부분 트리(subtree)역시 트리의 모든 특징을 따릅니다.
- 연결된 트리 내에서는 임의의 두 노트를 이어주는 경로는 유일하다.

### 트리의 핵심 이론
트리 관련 문제를 해결하려면 먼저 트리의 구성 요소에 관해 간단하게 알아볼 필욕 있습니다.
